Para fazer esse HTML, assisti as aulas e me aprofundei no assunto externamente, segue abaixo algumas tags e como ultilizá-las:

Tags, definição e exemplos

^topo^

Observação: sempre que for ultilizado o caracter "(" ou ")", refere-se ao caracter "<" e ">" respectivamente. E os caracteres "[" e "]" são irrelevantes quando colocados no código, estando aqui meramente para melhor visualizção.

  • As tags h [h1; h2; h3; h4; h5; e h6]

     As tags h são um conjunto simples de tags que definem o tamanho da fonte a ser exibida, onde h1 é a maior fonte, ultilizada principalmente em títulos, h2 um pouco menor que h1 ultilizada principamente em subtítulos, h3 menor que a h2 e assim sussecivamente até a h6, que é a menor entre as h

    Ulttiliza-se assim: [(h1) texto (/h1)] que resultará em uma fonte de maior tamanho

  • A tag p

     A tag p é ultilizada para demarcar um parágrafo, caso o programador deseje pular uma linha e criar outro paragrafo, deverá fechar essa tag e abrir uma nova.

    É usada assim:

    [(p)[...](/p)] criando dessa forma um parágrafo

  • A tag mark

     Essa tag é ultilizada para marcar e testacar um trecho de um texto.

    Por exemplo:

    [(mark) Exemplo (/mark)] dando esse retorno a baixo:

    Exemplo

  • A tag small

     usa-se para deixar um texto menor.

    É escrita assim:

    [(small)exemplo(/small)] dando esse retorno:

    exemplo

  • As tags i, u e strong

     Essas são tags usadas para formatar texto, a tag "i" é ultilizada para deixar o texto em itálico, a tag "u" sublinha o texto, ja a tag "strong" deixa o texto em negrito.

    Exemplo usando a tag "i" e é escrita assim: [(i)Exemplo(i/)]

    Exemplo usando a tag "u" e é escrita dessa forma: [(u)Exemplo(/u)]

    Exemplo usando a tag "strong" sendo escrito assim: [(strong)Exemplo(/strong)]

  • As tags "ol","ul" e li"

     Essas são tags de listas, onde "ol" é uma lista que segue uma ordem numérica, e "ul" é uma lista desordenada, que pode ser usada como tópicos. Li é uma tag usada ou dentro de "ol" ou de "ul" que diz o que vai ser escrito em cada topico, ou lista.

    1. exemplo
      2. Que é escrito da seguinte forma:
      [(ol)
      (li)
      exemplo
      (li)
      (ol)]

    • exemplo
      • Que é escrito dessa forma:
      [(ul)
      (li)
      exemplo
      (li)
      (ol)]

    Caso o programador queira inserir mais tópicos, basta inserir mais tags (li).

  • A tag "a"

     É usada para criar um link para uma página externa, ou ancoragem dentro da mesma página.

    Esse exemplo leva ao topo da página

    Sendo o seu código assim:

    [(p)(a href="#top")exemplo(/a)(/p)

    Sendo que nesse caso ja foi feita a ancoragem da palavra "top" com o topo da página

    Você pode connferir um vídeo explicando como é feita a ancoragem ja citada clicando AQUI.

  • A tag "hr"

     Essa tag define uma quebra temática em uma página HTML, como uma mudança de tópico, e é usada para separar o conteúdo em uma página HTML.

    exemplo

    Podendo ser escrita de forma simples:

    (hr)
    (p)exemplo(/p)
    (hr)

  • As tags "Sub" e "Sup"

     A tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. Podem ser usados para fórmulas químicas ou matemáticas.

    Exemplo: H2 Escrevendo [H(sub)2(/sub)O]

    Exemplo: X 2 Dessa forma: [X(sup)2(/sup)]

  • A tag "blockquote"

    É usado para exibir citações de outros sites

    Exemplo:

    Viver não é esperar a tempestade passar, é aprender a dançar na chuva. - Site "O pensador"

    Foi usado desta forma: [(mark)texto a ser marcado(/mark)

Novas tags HTML

^topo^

Como existem múitas tags HTML, e as aulas da DIO tem um tempo limitado, seria impossível apresentar todas tags nas aulas do Bootcamp, portanto, em busca de descobrir mais funcionalidades e possbilidades do HTML, me dispus a pesquisar mais sobre outras tags, e aqui apresento algumas:

  • A tag "font"

    A tag foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto, não é mais suportado em HTML5, agora se usa a tag "style".

    Exemplo

    Usando: [(p style="color:brown")exemplo(/p)

    Exemplo

    Dessa forma: [(p style="color:rgb(59, 59, 255)")Exemplo(/p)]

  • A tag "del"

    Essa tag serve para marcar o texto como uma parte exluida, porém continuar exibindo

    Texto Apagado

    Sendo assim: [(p)Texto (del)Apagado(/del)(/p)]

  • A tag "abbr"

    Essa tag serve para abreviar palavras nomes de pessoas, etc. A descrição aparece em um pequeno balão quando o mouse fica em cima da palavra. Exemplo:

    WWW

    Logo: [(p)(abbr title="Word Wide Web")WWW(/abbr)(/p)]
